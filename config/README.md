# Configuration

This directory contains configuration files for the application.

## Files

- `.env.example` - Example environment variables (copy to `.env` and fill in your values)
- Additional config files as needed

## Environment Variables

See `GETTING_STARTED.md` for instructions on setting up your `.env` file.

Key variables:
- `DATABASE_URL` - Database connection string
- `BLS_API_KEY` - BLS API key (if required)
- `DEBUG` - Debug mode flag
- `SECRET_KEY` - Application secret key
