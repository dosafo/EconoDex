# Documentation

This directory contains project documentation.

## Structure

```
docs/
├── api/               # API documentation
├── data_sources/      # Data source documentation
├── architecture/      # System architecture docs
└── guides/           # User and developer guides
```

## Documentation to Create

1. **API Documentation**: Endpoint specifications, request/response formats
2. **Data Source Documentation**: Available data sources, collection methods
3. **Architecture Documentation**: System design, data flow
4. **Developer Guide**: Setup instructions, contribution guidelines
5. **User Guide**: How to use the platform
